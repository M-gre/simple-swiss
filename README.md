# Swiss Tournament

 A single-page Swiss-style tournament runner. Go to [m-gre.github.io/simple-swiss](https://m-gre.github.io/simple-swiss/) to use it.

 ## Run locally
 Open `index.html` in a browser.

 ## Features
 - Configurable rounds and Buchholz tiebreaker (full / cut-1 / median)
 - Automatic Swiss pairing with rematch avoidance and byes for odd counts
 - Live standings with W-D-L record and Buchholz column
 - Drop / re-add players mid-tournament
 - JSON export & import for backup
 - Print-friendly stylesheet
